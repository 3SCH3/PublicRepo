# SinCos Arena

Kleines Browser-Game (Top-Down Shooter, inspiriert von Brawl-Gameplay), gebaut mit einfacher HTML/CSS/JavaScript.

## Features

- Touch-Steuerung fuer iPad (links bewegen, rechts zielen/schiessen)
- Nach jeder Welle: Lernrunde mit Aufgaben zu `sin`, `cos`, `tan` im rechtwinkligen Dreieck
- Fokus auf Verstaendnis: `sin` und `cos` sind Verhaeltnisse, keine absoluten Laengen
- Bessere Grafik mit Partikeln, Props und UI-Textur
- Keine Build-Tools, keine Dependencies

## Lokal starten

Einfach `index.html` im Browser oeffnen.

## Public Repo teilen

Da es nur statische Dateien sind, kannst du es direkt ueber GitHub Pages veroeffentlichen:

1. Repo auf GitHub pushen
2. In GitHub: `Settings -> Pages`
3. Source: `Deploy from a branch`
4. Branch: `main` und Ordner `/ (root)`
5. Speichern, dann den GitHub-Pages-Link teilen

Dateien:

- `index.html`
- `style.css`
- `game.js`
- `assets/kenney-mobile-controls-sample.png`
- `assets/kenney-mobile-controls-preview.png`
